# PyDataの行動規範
PyDataはNumFOCUSの教育プログラムであるため、  NumFOCUSの行動規範を順守します。

```
行動規範インシデントを報告する
```

PyDataコミュニティの各メンバーの参加を大切にし、すべての参加者に楽しく充実した体験をしてもらいたいと考えています。したがって、すべての参加者は、PyDataが公式に後援しているかどうかにかかわらず、会議全体およびすべての会議イベントで他の参加者に敬意と礼儀を示すことが期待されます。


PyDataはNumFOCUSプログラムであるため、何が期待されるかを明確にするために、PyDataイベントのすべての代表者、講演者、出展者、およびボランティアは、NumFOCUS行動規範に準拠する必要があり ます。主催者は、イベント全体を通じてこのコードを施行します。

ここでは、対面イベントに特に関連するNumFOCUS行動規範の特定の部分を再現します。すべてのPyData参加者は、完全な行動規範を確認することをお勧めします 。

## 短いバージョン
他の人に親切にしてください。他人を侮辱したり、下ろしたりしないでください。専門的に行動します。PyDataには、嫌がらせや性差別、人種差別、または排他的なジョークや言葉遣いは適切ではないことに注意してください。

すべてのコミュニケーションは、さまざまなバックグラウンドを持つ人々を含むプロの聴衆にとって適切なものでなければなりません。性的な言葉や画像は適切ではありません。

PyDataは、性別、性的指向、性同一性、表現、障害、容貌、体の大きさ、人種、宗教に関係なく、すべての人にハラスメントのないイベント体験を提供することに専念しています。私たちは、いかなる形であれ、参加者への嫌がらせを容認しません。

これをすべての人にとって歓迎的で友好的なコミュニティにするのを手伝ってくれてありがとう。

## 報告する方法
PyDataイベントで報告をしたい場合は、イベントスタッフのメンバーを見つけて、PyData委員会の委員長に相談してください。 議長と話をしたくない場合、または何らかの理由で議長が利用できない場合は、別のPyData委員会メンバーがあなたの報告を受け取ります。

通常、PyDataイベントのスタッフは、そのように識別するシャツを着ています。登録テーブルまたはNumFOCUSブースで、PyDataスタッフと話すように依頼することもできます。

PyDataのミートアップで、ミートアップの主催者の1人と話すように依頼します。

NumFOCUSは、オンサイトのイベントスタッフに連絡するだけでなく、NumFOCUS行動規範施行チームに送信されるレポートフォームを使用してレポートを作成することもお勧めします 。

イベントやミートアップでインシデントが発生した場合、NumFOCUS行動規範施行チームは、インシデントをフォローアップするために、必要に応じて関連する主催者/コミュニティマネージャー/プロジェクトリーダーに連絡します。

## イベントでの報告
NumFOCUSイベントに参加していて、レポートを作成したい場合は、イベントスタッフ/ミートアップ主催者に連絡して、 適切な迅速な対応ができるようにしてください。

さらに、ここでNumFOCUSにレポートを提出して ください。

イベントスタッフ/交流会の主催者は、参加者が会場のセキュリティや地元の法執行機関に連絡したり、護衛を提供したり、その他の方法でハラスメントを経験している人がイベントの期間中安全であると感じるのを支援します。

イベント固有の報告手順は、イベントのWebサイトにあります。Meetup固有のレポート手順は、Meetupグループのページにあります。 報告の指示は、イベント/ミートアップ自体で発表および/または投稿する必要があります。

タイムリーかつ効果的な方法で行動規範違反に対処する当社の能力は、提供できる情報の量に影響されるため、レポートフォームでは、次の情報をできるだけ多く含めるように求められます。

- **あなたの連絡先情報** （フォローアップが必要な場合に連絡できるようにするため）。これは秘密にされます。匿名を希望する場合、最初の報告を受け取った人以外にあなたの情報が共有されることはありません。
- インシデントの おおよその時間と場所 （できるだけ具体的にしてください）
- 行動が報告されている個人の識別情報（名前、ニックネーム、スクリーン名、身体的説明など）
- 行動の説明 （嫌がらせの言葉を報告する場合は、使用されている言葉を具体的に記入してください）、 何が起こったかのアカウント、および利用可能な サポート記録 （電子メール、GitHubの問題、スクリーンショットなど）
- インシデントを取り巻く状況/コンテキストの説明
- インシデントが進行中であるかどうか、および/またはこれが進行中の行動パターンの一部であるかどうかをお知らせください 
- この事件を目撃した、または関与した他の人の名前と連絡先情報（可能であれば）。（他の誰かが事件を観察しましたか？）
- 私たちが持つべきだとあなたが信じるその他の関連情報

イベントスタッフ/交流会の主催者は、口頭で報告する人から上記の情報を収集して書き留めようとします。報告に効果的に対応するためには、詳細を書面で記録することが非常に重要です。イベントスタッフが口頭で取られたレポートを書き留める場合、レポートを作成する人は、正確さのために書かれたレポートを確認するように求められます。

NumFOCUSおよび/または当社のイベントスタッフは、特に対面式のイベントにおいて、お客様の安全を確保し、差し迫ったニーズに対応するよう努めます。

NumFOCUSイベントでは、参加者が行動規範に違反する行動をとった場合、会議の主催者とスタ​​ッフは適切と思われる行動を取ることができます。

対面イベントでNumFOCUS行動規範に違反した場合の潜在的な結果には、次のものがあります。

- その人に彼らの行動をやめるように警告し、それ以上の報告は制裁につながることを警告します
- イベントの残りの期間、嫌がらせをしている人とのやり取りや物理的な近接を避けるように要求する
- ポリシーに違反する話を早期に終了する
- ポリシーに違反した講演のビデオまたはスライドを公開しない
- ポリシーに違反した講演者が、現在または将来のイベントで（さらに）講演することを許可しない
- 報告された人が保持しているイベントボランティアの責任と特権を直ちに終了する
- NumFOCUSが実行される将来のイベントにボランティアとして参加しないことを要求する（無期限または特定の期間）
- 払い戻しなしでイベントから人を追放する
- その人がすぐにイベントを離れ、戻ってこないことを要求する
- 将来のイベントからその人を禁止する（無期限または特定の期間）
- NumFOCUSが状況に必要かつ適切であるとみなすその他の対応
